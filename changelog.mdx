<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for shared documentation spaces.
- Introduced template gallery for quick project setups with the "Start from scratch" slogan.

## Improvements

- Enhanced search functionality across all documentation pages.
- Optimized loading times for large documentation repositories by `>30%`.

## Bug Fixes

- Fixed sync issues when editing docs across multiple browsers.
- Resolved export errors for Markdown files with special characters.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

## Bug Fixes

- Patched authentication token refresh failures during long sessions.
- Corrected table rendering issues in preview mode.

</Update>

## How to Report Issues

Help improve Next Step Conrad by reporting bugs or suggesting features.

<Steps>
  <Step title="Search Existing Issues" icon="search">
    Check the [issue tracker](https://github.com/nextstepconrad/next-step-conrad/issues) for similar problems.
  </Step>
  <Step title="Create New Issue" icon="plus">
    Click "New Issue" and select the appropriate template.
  </Step>
  <Step title="Provide Details" icon="edit-3">
    Include steps to reproduce, screenshots, and your environment details.
  </Step>
  <Step title="Submit and Follow Up" icon="send">
    Submit and watch for updates from the team.
  </Step>
</Steps>

<Callout kind="tip">
  For urgent issues, join the discussion on the [blog comments](https://nextstepconrad.com/blog) or contact support.
</Callout>
